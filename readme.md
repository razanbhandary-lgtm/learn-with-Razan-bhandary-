# PhishLab 🕵️

## What is this?
PhishLab is an educational tool for cybersecurity students to generate phishing links for testing.

## Features:
- Generate phishing links for various platforms
- Camera permission request on target device
- Captured data shows on dashboard
- Footer: **Developed by RAZAN Bhandary**

## How to use:
1. Enter Student ID and Name
2. Select a platform
3. Click "Generate Phishing Link"
4. Share the link with your test device
5. Target device opens link → camera permission asked
6. All captured data shows on dashboard

## ⚠️ Warning
- For educational purposes only
- Only test on your own devices
- Do not use for malicious purposes
